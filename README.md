-- Overview --
This is a basic website with a searchbar that allows the user to search for GIFs

--How to Use--
Once you have the HTML page loaded up, enter a keyword into the searchbar and press the submit button. This will display a set of 12 GIFs related to your keyword.

--Technologies Used--
This project uses HTML and CSS to create and format the website, JavaScript to add functionality to the searchbar, and Giphy to pull the GIFs from.

--Ideas for Future Improvement--
The first thing I think I could have done better with this website is mobile compatibility. I don't think I did enough formatting to ensure that the site properly displays on mobile devices, so I'll have to review Bootstrap breakpoints so I can properly implement this. I also think I could have improved on the design of the site, as I think it came out pretty basic. Next time, I'll have to try using images to improve the design a bit. I could also potentially add more pages to make the navbar functional, which was unnecessary for the project but could potentially improve the site.

--User Stories--
"As a GIF user, I want a site that displays various GIFs so that I can get GIFs more easily."
"As a user, I want a searchbar so that I can look through only the GIFs that match my needs."
"As a site manager, I want the framework for a navbar so that I can add additional pages in the future."
